## Summary

0.15.0 redefines `ai/memory/context.md`: from an append-only, model-facing context log (Standing context /
newest-first Log / Previous History) to a **detailed summary of the current session's context**, rewritten
**in place** at two save points - **before context compaction** (automatic or manual) and whenever the user
says "save/remember/update/retain/keep context" or similar. Per-turn logging stays in `interactions.jsonl`.
The template and `engineer.agent.md` (rev 16) carry the new semantics; a project's existing `context.md`
content is project-owned data and is converted by hand below. Low risk; nothing is lost if the pre-flight
backup is taken.

## Pre-flight checks

- [ ] `ai/manifest.json` exists and is valid JSON.
- [ ] The framework is at 0.15.0 (`uv run -m solaris.tools.version current`).
- [ ] **Back up the old log:** copy `ai/memory/context.md` to `ai/memory/context.md.bak` (the file is
      gitignored, so git history cannot restore it).

## Migrate

1. **Fast-forward the engineer agent:** `uv run -m solaris.tools.revs ff --dir <project>` picks up
   `engineer.agent.md` rev 16 when the project copy is unmodified. If it reports a merge, replace the
   old context-log paragraph in `## Memory` with the template's session-context paragraph by hand and
   bump the file's rev.
2. **Convert `ai/memory/context.md`** to the new template structure (`solaris/templates/ai-pack/ai/memory/context.md`):
   restructure to the intro + `## How to use this file` + `## Session context` layout, and write
   `## Session context` as a condensed summary of the old `## Standing context` plus the most recent
   `## Log` entries (current state, decisions, findings, next steps).
3. **Relocate durable knowledge:** fold anything in the old `## Standing context` that is long-lived
   (code map, run/deploy recipe, gotchas) into `ai/engineer.instructions.md` (shareable) or
   `ai/memory/resources.md` (inventory) so it survives future in-place rewrites.
4. **Stamp the version:** `uv run -m solaris.tools.version set --dir <project> 0.15.0`.
5. Once satisfied, delete `ai/memory/context.md.bak`.

## Validate

- [ ] `ai/manifest.json` `framework_version` is `0.15.0`.
- [ ] `ai/memory/context.md` has a `## Session context` section and no `## Log` / `## Previous History`.
- [ ] `ai/engineer.agent.md` `## Memory` describes the session-context summary and its two save points.
- [ ] `uv run -m solaris.tools.version check --dir <project>` passes.

## Revert

1. Restore `ai/memory/context.md` from `ai/memory/context.md.bak`.
2. Restore the previous `ai/engineer.agent.md` (rev 15) from git history.
3. `uv run -m solaris.tools.version set --dir <project> 0.14.0`.
